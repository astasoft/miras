SSH Tanpa Password (Passwordless SSH)
=====================================
1. Generate SSH Key menggunakan algoritma enkripsi RSA

$ ssh-keygen -t rsa
Generating public/private rsa key pair.
Enter file in which to save the key (/home/username/.ssh/id_rsa): (LANGSUNG ENTER)
Enter passphrase (empty for no passphrase): (LANGSUNG ENTER)
Enter same passphrase again: (LANGSUNG ENTER)
Your identification has been saved in /home/git/.ssh/id_rsa.
Your public key has been saved in /home/username/.ssh/id_rsa.pub.
The key fingerprint is:
8c:89:e3:23:09:0a:a6:9a:5b:3b:d9:e5:ed:ad:ec:f6 git@astasoft
The key's randomart image is:
+--[ RSA 2048]----+
|                 |
|                 |
|                 |
|     . +         |
|o.  o o S        |
|=. o ..          |
|o +ooo .         |
|.ooo....o.       |
|+...   +=oE      |
+-----------------+

2. Kirim public key anda (/home/username/.ssh/id_rsa.pub) ke email 
   saya di me@rioastamal.net. Sebaiknya ikuti format berikut ketika mengirim email
   Subject: Public key <NAMA>
   Attachment: /home/username/.ssh/id_rsa.pub (direktori .ssh itu hidden jadi tekan CTRL+H untuk show hidden file saat browsing file)
   Body:
   - My Public Key -
   
   Contoh: 
   Subject: Public key Farid
   Attachment: /home/username/.ssh/id_rsa.pub
   Body:
   - My Public Key -

3. Saya akan memasukkan isi id_rsa.pub kedalam authorized user yang dapat push ke repo kolaborasi.
