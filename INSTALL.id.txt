INSTALASI MIRAS
---------------
Dimulai dari versi 1.0.5 instalasi otomatis via installer telah ditiadakan, 
metode yang tersedia hanya instalasi secara manual.

DAFTAR ISI
----------
1. Kebutuhan Sistem
2. Panduan Instalasi
3. Troubleshooting


1. Kebutuhan Sistem
===================
* Web Server dengan PHP Support
* PHP versi 5.2.0 dengan MySQLi extension
* MySQL 4.1+

2. Panduan Instalasi
====================
Bagi anda yang sudah familiar dengan tipikal instalasi berbasis AMP 
(Apache, MySQL, dan PHP) maka instalasi Miras juga tidak jauh berbeda. 
Hanya perlu 5 langkah yaitu:

  1. Extract Miras ke direktori web server (biasanya htdocs).
  2. Buat database baru pada MySQL.
  3. Import script SQL yang ada didalam direktori miras yaitu miras.sql.
  4. Edit file mr/db_config.php, sesuaikan dengan konfigurasi database anda.
  5. Buka http://localhost/miras/ dan instalasi pun selesai.

Lakukan tes berikut untuk memastikan Miras terinstal dengan sempurna.

  1. Klik menu Tugu Pahlawan
  2. Klik 'Dokumen Rahasia' (Akan muncul pesan error)
  3. Masukkan username admin dan password admin123
  4. Klik kembali 'Dokumen Rahasia'
  5. Jika anda berhasil melihat halaman tersebut maka proses autentikasi yang 
     melibatkan session berhasil.

3. Troubleshooting
==================
Pastikan konfigurasi database dan koneksi mysql tidak bermasalah. Jika 
menggunakan linux dan mengaktifkan cache, pastikan direktori mr/cache/query dan 
mr/cache/content dapat ditulis oleh Miras. Lakukan chmod 777 pada kedua 
direktori tersebut jika perlu.

Jika masih terdapat error pastikan langkah-langkah diatas sudah benar, namun 
jika sudah benar tapi tetep tidak bisa. Mungkin anda menemukan sebuah bug, 
silahkan posting masalah anda dimilis MIRAS dialamat:
   http://groups.google.com/group/miras-id 

atau posting ke Issue Tracker di:
   https://github.com/astasoft/miras/issues
