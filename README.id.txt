PENGENALAN
----------
Miras adalah sebuah PHP framework yang mengusung MVC pattern dengan pendekatan
secara prosedural.

VERSI
-----
Versi yang disertakan dalam distribusi adalah Miras v1.0.1. Mohon diingat, 
status dari versi ini adalah experimental untuk itu tidak disarankan digunakan
pada live production.

DOKUMENTASI
-----------
Dokumentasi lengkap miras dapat dilihat pada halaman
http://miras.raweet.org/index.php/documentation
   atau
dapat dilihat pada direktori docs/

INSTALASI
---------
Instrukti instalasi dapat dilihat pada
http://miras.raweet.org/index.php/install
   atau
pada file INSTALL.id.txt
